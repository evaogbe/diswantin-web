# Diswantin Web

The web app for Diswantin, a productivity app that shows you the one thing to do
right now.

## Development

### Requirements

- [NodeJS](https://nodejs.org)
- [Docker](https://www.docker.com)

### Setup

To setup the local postgres database run:

```sh
docker-compose up -d
npm run db:push
```

### Dev server

```sh
npm run dev
```
